# Week 2 Learning Journal

## Week Learning Activities & Resources

This week I learned how to install and use **Joomla CMS**. I created a free hosting account using InfinityFree and installed Cholla 6.1.0. After installation, I explored the Joomla administrator dashboard and familiarised myself with its core components, including the Article Manager, Menu Manager, and Module Manager.
I spent time learning how to manage content in Joomla by creating articles and organising them into categories and learned how to make them appear on the website using menu items. I also practiced creating new menus, adding menu items and assigning articles to them via the Menu Manager. Additionally,  I also researched how to insert custom HTML code in source code mode into a DOM article to embed embedded CSS styles into the content area.

I also observed several websites for design inspiration, including Oclean, Oral-B, Philips Sonicare, and Colgate. Although I currently cannot reach their professional level, I learned a lot from their product layouts, visual hierarchy, and call‑to‑action placements. These observations helped me improve my own SmileBrush website pages (Home, Products, About) by making the layout cleaner and more user‑friendly.

## Content Insights

Joomla separates **content** (articles) from **presentation** (menus). This allows me to reuse content across different pages easily. I learned that using “Single Article” menu items is the simplest way to display custom HTML pages. Comparing Joomla to WordPress, I found Joomla has a steeper learning curve but offers greater control, especially for structured content and multilingual sites – useful knowledge for future projects.

I created three Joomla articles (Home, Products, About) and using the Menu Manager, I created a main menu for my site, added three menu items, set each menu item to “Single Article” type, and linked each one to the corresponding article.I used a Custom HTML Module to inject global CSS styles across all pages (though for simplicity I also embedded styles directly in each article).
