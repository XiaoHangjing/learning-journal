## Content Insights

This week helped me better understand the purpose of local development.

I was a little confused about why we needed to use Local WP when I already had a website online. After working through the practical activity, I realised that local development allows developers to build and test websites safely before publishing them.This is in line with one of the three methods taught by the teacher in class

I created and managed pages through the WordPress dashboard and became more familiar with how WordPress handles website content. Compared with building a static HTML website, WordPress makes it much easier to update content without editing code every time.

One thing I learned was that websites created in Local WP can only be accessed on my own computer. This is different from a hosted website, which anyone can visit through a web link. Understanding the difference between a local environment and a live environment helped me understand how professional websites are developed.

I also spent some time checking my website structure and making sure all three pages were organised correctly before moving on to deployment.
